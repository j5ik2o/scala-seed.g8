# A Giter8 template for Scala seed project.

Create a new project to support followings using giter8:

- Scala seed files.
- Compile with Formatting by Scalafmt

## preparing

```sh
$ brew install giter8
```

## usage

- for giter8

```sh
$ g8 j5ik2o/scala-seed.g8
```

- for sbt new

```sh
$ sbt new j5ik2o/scala-seed.g8
```
